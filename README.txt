نظام إدارة المخازن - الجيش المصري
====================================

متطلبات التشغيل:
-----------------
1. Delphi 10.3 أو أحدث (Community Edition مجاني)
2. Windows 7/8/10/11
3. مكتبة FireDAC (متضمنة في Delphi)

خطوات البناء:
--------------
1. افتح ملف WarehouseSystem.dpr في Delphi
2. اختر Project > Build WarehouseSystem
3. للحصول على ملف exe مستقل:
   - Project > Options
   - Building > Delphi Compiler > Linking
   - تأكد من تفعيل "Link with runtime packages" = False
   - تأكد من تفعيل "Generate console application" = False

4. البناء النهائي:
   - Project > Build WarehouseSystem
   - الملف التنفيذي سيكون في مجلد Win32\Debug أو Win32\Release

البيانات الافتراضية:
--------------------
كلمة المرور الافتراضية: qwer56qwer

قاعدة البيانات:
---------------
- يتم إنشاء قاعدة بيانات SQLite تلقائياً (warehouse.db)
- يتم حفظها في نفس مجلد البرنامج
- لا تحتاج تثبيت إضافي

مميزات البرنامج:
----------------
1. تسجيل دخول آمن بكلمة مرور
2. البحث عن قطع الغيار برقم القطعة
3. سحب الكميات من المخزن
4. إضافة قطع جديدة
5. طباعة إيصالات السحب
6. واجهة عربية بالكامل مع محاذاة من اليمين

التوزيع:
--------
لتوزيع البرنامج، انسخ الملفات التالية:
1. WarehouseSystem.exe
2. warehouse.db (سيتم إنشاؤها تلقائياً عند أول تشغيل)

ملاحظات للمطور:
----------------
- جميع النصوص بالعربية
- BiDiMode = bdRightToLeft للمحاذاة من اليمين
- استخدام خط Arial للدعم الجيد للعربية
- الأرقام والتواريخ تستخدم BiDiMode = bdLeftToRight
